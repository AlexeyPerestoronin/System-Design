# Desktop Editor

Создать редактор документов, обладающий универсальной гибкостью адаптации к:
1) Работе с документами любого рода:
    Редактор не должен налагать какие-либо ограничения на возможность работы с документами любых типов (текст, таблицы, презентации, картинки, музыка, и т.д.), но предоставлять возможности настройки для работы с ними.
2) Настройке пользовательского интерфейса:
    Редактор не должен обязывать любых своих пользователей привыкать к своему рабочему интерфейсу, но должен предоставлять возможность гибкой настройки последнего в согласии с пожеланиями пользователей.
3) Пользователи всех масштабов (обычные, продвинутые, B2B):
    Редактор (в контексте всей Системы) должен быть универсальной платформой для решения самых необычных задач,как рядовых пользователей так и бизнес-пользователей, в том смысле, что решение задачи в Редакторе - это, во-первых, процесс создания рабочей среды (настройка Редактора на визуальном и структурном уровнях), и, во-вторых, собственно, процесс работы с целевым документом.
